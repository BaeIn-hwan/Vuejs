vue js 세팅방법
1. node.js LTS 버전 다운로드
2. 크롬에서 Vue.js devtools 설치
3. Vuejs 폴더에서 커맨드 창 켜서 npm으로 Vue-CLI 설치하기
npm install -g yarn @vue/cli (windows)
sudo spm install -g yarn @vue/cli (macOS)
